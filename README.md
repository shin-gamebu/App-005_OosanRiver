# App-005_OosanRiver

野生のオオサンショウウオを眺める静かな癒し系Webアプリ

## 概要

このアプリは、野生のオオサンショウウオを眺めるだけの静かなWebアプリです。
- ユーザーは「育てない」、世話は最小限
- 成長は非常にゆっくりで、一生続く
- 放置すると元気がなくなり、さらに長期間放置すると姿が消える
- 責める表現や煽りは一切使わない

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

## 使い方

- 画面をクリックすると、低確率でオオサンショウウオが小さく反応します
- 毎日開くと、ゆっくりと成長します
- 3日以上開かないと元気がなくなります
- 7日以上開かないと姿が消えます
- 画面下部に日次ログが表示されます

## 技術スタック

- React 18
- TypeScript
- CSS3
- localStorage（データ永続化）

## ビルド

```bash
npm run build
```
